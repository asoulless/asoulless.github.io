The current website you’re on right now! This one was made using Zola and Daudix’s Duckquill theme. Originally, I was thinking of using his Amertrine theme, but that one’s still in its pre-alpha state (and I couldn’t figure out how to fix the issues I was having with it when I did attempt to use it fast enough lol), so I just settled on Duckquill. I think the website’s leagues better than the old one I had. It’s now hosted on GitHub Pages for now, though I would like to find a better place to host it in the future. For now, though, I just want to see how expansive this site will get…
